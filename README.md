## 开发版特点

- 开发版使用和正式版相同的更新检测渠道，因此无法获得最新开发版，需要更新，请手动监察是否更新
- 开发版命名规则：发布版本号+Build内部版本号+Dev+第几次提交
- 开发版是开发者在云端开发环境中进行开发，测试能够运行后才提交至 Dev 分支，因此通常情况下，开发版是可以正常运行并使用的
- 开发版在正式版发布前通常未经过代码测试，不保证百分百稳定
